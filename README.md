# 蜡笔公众号排版器

一个面向微信公众号创作者的单页排版工具。左侧编辑文章，右侧实时预览，最后一键复制到公众号后台，适合把草稿、Markdown 或普通长文快速整理成稳定的公众号富文本样式。

## 核心功能

- 双栏编辑与实时预览，支持手机宽度预览。
- Markdown 解析：标题、加粗、斜体、引用、列表、代码块、分割线。
- 智能预处理：保守整理、长文分层、教程步骤、种草清单、口播转图文。
- 8 套排版模式：教程干货、深度长文、知识博主、日系文艺、甜系种草、品牌商务、马里奥主题、轻手账咖啡馆。
- 低饱和预设配色、自定义配色，以及 3 个本地自定义配色槽。
- 标题、引用、列表、提示卡、结尾组件等会跟随配色联动。
- 图片支持本地上传、URL、剪贴板粘贴；单图、双图并排均可用，单张图片限制 10MB。
- 粘贴到编辑区的图片可手动拖到另一张图上形成并排，并排图内可互换位置。
- 支持文字高亮、编号标题、提示卡片、文章结尾、设计排版组件。
- 内容暂存、一键清空、保存我的样式、导出 HTML、复制为公众号兼容 HTML。
- 一键复制到公众号：复制前自动做内联样式与公众号兼容处理。

## 使用方式

在线访问：

```text
https://crayonpb.vercel.app
```

本地使用：

1. 下载或克隆仓库。
2. 双击打开 `index.html`。
3. 在左侧输入或粘贴文章，右侧查看预览。
4. 选择排版模式、字体、配色、背景和间距。
5. 插入图片、提示卡、编号标题或结尾模块。
6. 点击「一键复制到公众号」，到公众号后台正文区粘贴。

无需安装依赖，也不需要启动服务器。

## 本地存储

本工具没有账号系统。以下内容会保存在当前浏览器本地：

- 我的样式
- 内容暂存
- 3 个自定义配色槽

换电脑、换浏览器或清理浏览器数据后，这些本地数据不会自动同步。

## 注意事项

- 复制到公众号后台后，建议再预览一次，公众号编辑器可能会二次处理样式。
- 本地图片会以 Base64 进入文章内容，图片越大，复制和编辑越吃力。
- 外链图片请确保地址长期可访问。
- 新增组件优先采用简单内联样式，减少公众号后台兼容问题。
