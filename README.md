# 蜡笔公众号排版器

一个面向微信公众号创作者的在线文章排版工具。它把编辑区、实时预览、样式配置和一键复制整合在一个页面里，适合把草稿、Markdown 文本或普通长文快速整理成适合公众号后台粘贴的富文本样式。

## 主要功能

- 实时双栏编辑：左侧编辑正文，右侧实时预览公众号排版效果。
- Markdown 自动解析：支持标题、加粗、斜体、引用、列表、代码块、分割线等常用语法。
- 智能预处理：支持保守整理、长文分层、教程步骤、种草清单、口播转图文等文章类型，把普通纯文本整理成更清晰的 Markdown 结构。
- 多套排版模式：内置教程干货、深度长文、知识博主、日系文艺、甜系种草、品牌商务、马里奥主题、轻手账咖啡馆等风格。
- 字体设置：可分别设置标题字体和正文字体。
- 配色系统：支持预设配色和自定义主色、辅色、强调色、正文色、背景色。
- 统一配色联动：标题装饰、引用块、列表编号、提示卡片、结尾组件等会跟随当前配色更新。
- 背景纹理：支持纯白、信纸、细格子、牛皮纸、薄荷、薰衣草、暖阳笔记、雾白细点、云蓝微格、奶油宣纸、淡粉斜纹等背景。
- 图片插入：支持本地图片、图片 URL 和直接粘贴剪贴板图片，并可设置宽度、对齐、圆角、边框；两张横图可选择「双图并排」一起插入。
- 高级组件：可插入文字高亮、编号标题、提示卡片、文章结尾模块。
- 设计排版组件：支持插入品牌实战风格的开头总结框、设计小标题和设计结尾。
- 手机预览：可切换到手机宽度预览排版效果。
- 样式管理：可保存、改名、删除、导出和导入自己的排版样式。
- 导出 HTML：可将排版后的文章导出为独立 HTML 文件。
- 一键复制到公众号：自动内联样式，复制后可直接粘贴到微信公众号后台编辑器。
- 本地安全加固：粘贴内容、导入样式、图片 URL、复制/导出内容会进行基础安全过滤。

## 如何使用排版器

### 方式一：在线使用

如果仓库已开启 GitHub Pages，可以直接访问项目页面：

```text
https://dorlarosendo434-hub.github.io/GZHcomposing/
```

如果暂时打不开，请到 GitHub 仓库的 `Settings -> Pages` 中开启 GitHub Pages，选择：

```text
Branch: main
Folder: /root
```

保存后等待一两分钟，再访问上面的链接。

### 方式二：本地使用

1. 打开本仓库。
2. 下载或克隆项目到电脑。
3. 双击打开 `index.html`。
4. 开始编辑文章。

无需安装依赖，也不需要启动服务器。

## 公众号文章排版步骤

1. 在左侧编辑区输入文章内容，或从文档、备忘录、Markdown 编辑器中粘贴内容。
2. 如果是普通纯文本，可以点击工具栏里的「预处理」，按文章类型选择保守整理、长文分层、教程步骤、种草清单或口播转图文，让工具自动整理标题、段落、列表和引用。
3. 点击顶部「排版模式」，选择适合文章气质的预设风格。
4. 点击「字体」，分别调整标题字体和正文字体。
5. 点击「配色」，选择预设配色，或自定义主色、辅色、强调色、正文色和背景色。
6. 点击「行距/段距」，调整阅读节奏。
7. 点击「背景」，选择文章背景纹理。
8. 使用工具栏插入图片、引用、列表、分割线、编号标题、提示卡片或结尾模块。插入图片时可选择「单张图片」或「双图并排」，也可以直接把 10MB 以内的图片粘贴到编辑器。
9. 如果文章是教程、案例拆解或品牌实战内容，可以点击「设计」插入开头总结框、设计小标题和设计结尾。
10. 点击「手机预览」，检查移动端阅读效果。
11. 确认无误后，点击底部「一键复制到公众号」。
12. 打开微信公众号后台文章编辑器。
13. 在正文编辑区按 `Ctrl + V` 粘贴。
14. 在公众号后台继续补充封面、摘要、原创声明等发布信息。

## 样式复用

如果你调出了一套常用风格，可以点击「保存为我的样式」。保存后可在「我的样式」中快速复用。

也可以点击「导出样式JSON」备份样式，或分享给别人。别人拿到 JSON 后，点击「导入样式JSON」即可使用同一套排版配置。

## 注意事项

- 复制到公众号后台后，建议再预览一次，因为公众号编辑器可能会对部分样式做兼容处理。
- 如果图片使用外链 URL，请确保图片地址长期可访问。
- 如果使用本地图片插入，图片会以 Base64 形式进入文章内容，图片过大可能影响复制和编辑体验。
- 样式数据保存在当前浏览器本地存储中，清理浏览器数据可能会删除已保存样式。
